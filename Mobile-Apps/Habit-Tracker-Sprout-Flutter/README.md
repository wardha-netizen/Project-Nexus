##  Project Overview
"Sprout" is a modern mobile application developed during my 6-week internship at **Rayonix Solutions**. It addresses the psychological challenge of habit formation by using streak-based motivation, visual analytics, and a clean user interface.



##  Key Features
* **Streak-Based Motivation:** Tracks consecutive days to encourage consistency.
* **Visual Habit Analysis:** Dashboard showing total habits vs. daily completions.
* **Custom Habit Frequency:** Options for Daily, Weekly, or Custom intervals.
* **Leaderboard Logic:** Displays the longest streaks for different activities (e.g., Exercise, Reading).
* **Local Storage/Firebase:** Implemented for secure data persistence.

##  Tech Stack & Implementation
* **Framework:** Flutter (Cross-platform Android/iOS)
* **State Management:** Followed Agile sprint methodologies for feature rollouts.
* **UI/UX:** Designed with "Sprout" branding (Green theme) to symbolize growth.
